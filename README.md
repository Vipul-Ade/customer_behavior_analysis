# customer_behavior_analysis - End-to-End Data Analytics Project

This project follows a structured end-to-end analytics workflow from understanding the business scenario and defining the business problem, to data preparation, SQL-based business analysis, interactive Power BI visualizations and business recommendations

Tools & Technologies Used:
✅ Python (Pandas) – Data Cleaning, Preparation & Feature Engineering
✅ SQL – Business Analysis by answering business questions
✅ Power BI – Interactive Dashboard & Data Visualization
✅ Gamma AI – Project Report
